# TCP vs UDP
#Główna różnica

TCP = reliable (pewne dostarczenie danych)  
UDP = fast (szybkie, ale bez gwarancji)

## 📦 TCP (Transmission Control Protocol)

✔ connection-oriented (najpierw połączenie)  
✔ checks delivery (sprawdza czy dane dotarły)  
✔ resends lost data (wysyła ponownie)  
✔ slower but safe  

👉 używane w:
- HTTP/HTTPS (strony www)
- email
- logowanie, banki

-

## ⚡ UDP (User Datagram Protocol)

✔ connectionless (bez połączenia)  
✔ no guarantee delivery (może zgubić pakiety)  
✔ very fast  
✔ no retransmission  

👉 używane w:
- gry online
- VoIP (rozmowy)
- streaming

-

## 🧠 Jak zapamiętać

TCP = T jak “Trust” (ufasz, że dojdzie)  
UDP = U jak “Unstable / Ultra fast”

## 💥 Jedno zdanie

TCP = slow but reliable  
UDP = fast but lossy
